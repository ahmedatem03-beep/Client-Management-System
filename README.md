# 👥 Client Management System (C++)

A console-based **Client Management System** developed using **C++** to manage client records efficiently.

The application allows users to add, update, delete, search, and display client information through a simple and interactive console interface.

---

## 📌 Features

- ➕ Add new clients.
- 📋 Display all clients.
- 🔍 Search for a client.
- ✏️ Update client information.
- ❌ Delete client records.
- 💾 File Handling for data persistence.
- 🖥️ Simple and user-friendly console interface.

---

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Visual Studio
- Windows Console
- Standard Template Library (STL)
